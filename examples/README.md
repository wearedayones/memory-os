# examples/

Example usage for hermes-memory.

- `examples/add_memory.py` - create a memory entry and commit via Git wrapper.
- `examples/search.py` - query memories with filters and tags.
- `examples/graph_demo.py` - create nodes and edges under graph rules.
- `examples/bootstrap_demo.py` - show how a fresh Hermes agent auto-bootstraps memory.

This scaffold intentionally contains no Python files yet. Future subagents should add runnable scripts here.
